# blog-tech-tech

## Description

To build a CMS- style blog that is similar to a Wordpress site.Developers can publish blog posts, and comment on other posts.

## Installation

To install dependencies, run the following command:

```
npm  install:
express
node
heroku
sequelize
mysql-2
express-handlebars
init
dotenv
bcrypt
express-session
connect-session-sequelize
```
